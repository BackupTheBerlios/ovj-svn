Generelle Daten


Hier werden die Daten des Distriktspeilreferenten eingetragen sowie zwei Dateien angegeben:
  1. Die PM Datei des Vorjahres (weitere Informationen)
  2. Die aktuelle PM Datei (weitere Informationen)
Es empfiehlt sich, diese Dateien im Programmverzeichnis von OVJ abzulegen, d.h. auf der obersten Verzeichnisebene.
Wenn einer der Buttons links neben den Eingabefeldern gedrückt wird, so öffnet sich ein Dateidialog, um die entsprechende Datei auszuwählen, deren Namen dann in das entsprechende Feld kopiert wird. Man kann das Feld natürlich auch von Hand ausfüllen.

Laden, Speichern und Importieren der Generellen Daten:
  • Über den Menupunkt 'Neu' im Datei-Menu werden die Eingabefelder gelöscht und.
  • Über den Menupunkt 'Öffnen' im Datei-Menu werden die Generellen Daten geladen.
    Die Generellen Daten befinden sich in einem projektspezifischen Verzeichnis unterhalb des Verzeichnisses 'config' unterhalb des Programmverzeichnisses und haben den gleichen Namen wie dieses Verzeichnis, so liegen z.B. die Generellen Daten 'Demo.txt' im Unterverzeichnis 'config\Demo'.
    Ein Projekt definiert sich über die jeweiligen Generellen Daten. Für jedes Projekt müssen in den übrigen Verzeichnissen die projektspezifischen Unterverzeichnisse vorhanden sein bzw. werden diese von OVJ bei Bedarf angelegt.
    Beim Starten von OVJ werden die zuletzt verwendeten Generellen Daten automatisch geladen (die Information darüber wird in der Datei 'ovj.ini' im Programmverzeichnis gespeichert).
  • Mittels des Menupunkt 'Importieren' können Generelle Daten z.B. aus dem Vorjahr importiert werden, dabei werden jedoch nur diejenigen Daten importiert, welche noch aktuell sind.
  • Über den Menupunkt 'Speichern' werden die Generellen Daten gespeichert.
Die Liste der OV Wettbewerbe wird zusammen mit den Generellen Daten in der gleichen Datei gespeichert.
