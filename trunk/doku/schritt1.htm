Die nachfolgenden Hinweise basieren auf der Annahme, dass die OVJ Software installiert und OVJ gestartet wurde,
entweder als OVJ.exe oder OVJ.pl über einen Perl-Interpreter.

Generelle Daten


Hier werden die Daten des Distriktspeilreferenten eingetragen sowie drei Dateien angegeben:
  1. Die PM Datei des Vorjahres (weitere Informationen)
  2. Die aktuelle PM Datei (weitere Informationen)
  3. Eine Datei zur Ersetzung von Vornamen, da oftmals Spitznamen wie Uli, Nick oder Didi in Auswertungen verwendet werden (weitere Informationen)
Es empfiehlt sich, diese Dateien im Programmverzeichnis von OVJ abzulegen, d.h. auf der obersten Verzeichnisebene.
Wenn einer der Buttons links neben den Eingabefeldern gedrückt wird, so öffnet sich ein Dateidialog, um die entsprechende Datei auszuwählen, deren Namen dann in das entsprechende Feld kopiert wird. Man kann das Feld natürlich auch von Hand ausfüllen.

Laden, Speichern und Importieren der Generellen Daten:
Über den Button 'Laden' werden die Generellen Daten geladen. Dabei wird die Jahreszahl verwendet, um den Dateinamen festzulegen: 'OVJ_Generell_XXXX.txt'
wobei XXXX durch die entsprechende Jahreszahl ersetzt wird. Die Generellen Daten befinden sich im Verzeichnis 'config' unterhalb des Programmverzeichnisses.
Beim Starten von OVJ wird dieses Verzeichnis automatisch nach 'OVJ_Generell_XXXX.txt'-Dateien durchsucht, die erste gefundene Datei wird geöffnet.
Daher muss der Button zum Laden normalerweise nie verwendet werden.

Über den Button 'Speichern' werden die Generellen Daten gespeichert. Dabei wird die Jahreszahl verwendet, um den Dateinamen festzulegen: 'OVJ_Generell_XXXX.txt'

Mittels des Buttons 'Importieren' können Generelle Daten z.B. aus dem Vorjahr importiert werden, dabei werden jedoch nur diejenigen Daten importiert,
welche noch aktuell sind.

Die Liste der OV Wettbewerbe wird zusammen mit den Generellen Daten in der gleichen Datei gespeichert.
