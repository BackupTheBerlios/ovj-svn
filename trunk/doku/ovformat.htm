Format einer Ergebnisliste zur automatischen Auswertung durch OVJ


Es gibt ein Format für den Kopfteil einer Ergebnisliste für eine OV Fuchsjagd, das von OVJ direkt unterstützt wird.
Dieses Format wird im Distrikt Oberbayern von Reinhard Hergert verwendet. Wenn Kopfdaten in diesem Format vorliegen,
kann damit automatisch der Kopfteil dieser OV Fuchsjagd in der OVJ-Oberfläche ausgefüllt werden.
Beispiel: 
Organisation:       OV Ermstal
Dok:                P31
Datum:              10.09.2006
Verantwortlich:     Marc Landmann, DL5SEZ, P31
Helfer:             Matthias Kühlewein, DL3SDO, P12
Teilnehmerzahl:     19
Band:               80m

Schlüsselwort Daten Kommentar
Organisation: Ortsverbandsname evtl. vorausgehendes Wort 'OV' mit anschliessenden Leerzeichen wird gelöscht.
Dok: DOK Groß-/Kleinschreibung bei Schlüsselwort und DOK egal.
Datum: Datum Datum Format wird teilweise geprüft.
Verantwortlich:
bzw.
Ausrichter: 		Verantwortlicher Ausrichter Format: Vorname Nachname Rufzeichen DOK Geburtsjahr
Dabei ist es egal, ob diese Wörter über Kommas oder ein oder mehrere Leerzeichen getrennt sind.
Teilnehmerzahl: Anzahl Teilnehmer Normalerweise wird die Anzahl der Teilnehmer bei der Auswertung ermittelt. Sobald jedoch im Feld 'Anz. Teilnehmer manuell' eine Zahl steht, wird diese stattdessen verwendet.
Helfer: Helfer Format: Vorname Nachname Rufzeichen DOK Geburtsjahr
Dabei ist es egal, ob diese Wörter über Kommas oder ein oder mehrere Leerzeichen getrennt sind.
Es können mehrere Helferzeilen vorhanden sein.
Band: Band ein oder zwei Zahlen

Die Helferdaten werden natürlich nicht in den Kopfteil des OV Wettbewerbs übernommen, sondern bei der Auswertung berücksichtigt.
Die Suche nach Kopfdaten wird abgebrochen sobald eine Zeile mit '---' beginnt.
Andere Schlüsselwörter werden ignoriert.
Bei den Daten für Helfer und Ausrichter können Angaben weggelassen werden, aber immer nur von rechts nach links.
Z.B. das Geburtsjahr und der DOK, aber z.B. nicht das Rufzeichen, wenn ein DOK angegeben ist, oder der DOK, wenn das Geburtsjahr angegeben ist.
