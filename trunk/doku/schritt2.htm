Liste der OV Wettbewerbe


In dieser Liste werden alle OV Wettbewerbe aufgeführt. Sie kann frei editiert werden.
Die Auswertung geschieht dann von oben nach unten.

Um einen OV Wettbewerb anzulegen, wird wie folgt vorgegangen:
  • Eingabe eines Namens. Dieser Name muss nicht identisch mit dem Dateinamen der Ergebnisliste sein.
  • Markieren des kompletten Namens, dies kann z.B. durch Doppelklick auf den Namen geschehen.
  • Button 'Editieren/Erzeugen' anklicken. Bereits existierende Wettbewerbsdaten werden geöffnet, ansonsten wird im Bereich OV-Wettbewerb mit leeren Datenfeldern begonnen.
Tip: Sollen Daten einer anderen OV Veranstaltung übernommen werden, da z.B. beide vom selben Veranstalter ausgerichtet wurden,
so wird zunächst der bereits angelegte OV Wettbewerb geöffnet. Dann wird in der Liste der OV Wettbewerbe eine neue Zeile mit einem
neuen Namen angelegt, dieser markiert und mit dem Button 'Erzeugen aus aktuellem OV Wettbewerb' eine Datenübernahme bewirkt.

Die anderen Buttons, welche die Auswertung betreffen, werden später erläutert.
