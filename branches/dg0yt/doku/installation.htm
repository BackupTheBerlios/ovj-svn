Installation

Systemvoraussetzungen

Entwickelt und getestet wurde OVJ unter Windows XP sowie Linux. Allerdings sollte es ebenfalls unter anderen Windows Versionen lauffähig sein, für die ein aktueller Perl Interpreter verfügbar ist, oder auf denen die gepackte .exe-Datei läuft.

Einleitung

Die Software ist in der Programmiersprache Perl geschrieben. Da diese standardmaessig auf Windows-Systemen nicht vorhanden ist, verwende ich das Perl-Modul PAR um die benötigten Dateien des Perl-Interpreters mit dem Programmcode zusammen in eine ausführbare exe-Datei zu verpacken.
(Ich erzeuge die ausführbare Datei mit der Anweisung 'pp -o OVJ.exe OVJ.pl' ).
Alternativ kann natürlich auch ein Perl-Interpreter installiert werden.

Verwendung mit Installation des Perl-Interpreters

Das ZIP-Archiv unter Beibehaltung der Ordnerstruktur entpacken.

Installation der Perl-Distribution von Activestate

Ich empfehle die Perl-Distribution von Activestate, diese kommt mit einem komfortablen Windows-Installer daher und ist umsonst.
Tip: die Felder auf der 'Contact Details' Seite sind optional, also einfach auf 'Continue' klicken.
Auf der nächsten Seite die MSI Version für Windows (x86) auswählen.

Ich habe die Software unter Perl 5.8.8.817 entwickelt, allerdings sollte auch jede neuere Version (und in gewissem Maße auch ältere Version) funktionieren.
Wer genau diese Version installieren will, hier ist der Link: http://downloads.activestate.com/ActivePerl/Windows/5.8/ActivePerl-5.8.8.817-MSWin32-x86-257965.msi
Hinweis: Für die Installation von Perl sind Administrator-Rechte notwendig.

Da die Perl-Installation die Dateiendung .pl automatisch mit dem Perl-Interpreter verknüpft, kann die OVJ-Software jetzt im Explorer durch Doppelklick auf 'OVJ.pl' gestartet werden.

Verwendung ohne Installation des Perl-Interpreters

Das ZIP-Archiv unter Beibehaltung der Ordnerstruktur entpacken.
Einfach die exe-Datei starten (dauert einige Sekunden), hierfür sind keine Administrator-Rechte notwendig.
