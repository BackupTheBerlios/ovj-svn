OV Wettbewerb


In diesem Bereich werden die Daten des jeweiligen OV Wettbewerbs angegeben.

Beim Klicken auf den Button 'Ergebnisliste' öffnet sich ein Auswahldialog, über den eine Ergebnisliste ausgewählt werden kann.
Die Ergebnislisten müssen sich im projektspezifischen Unterverzeichnis im Verzeichnis 'input' unterhalb des Programmverzeichnisses befinden (z.B. in 'input\Demo').

Sobald eine Ergebnisliste über den Button ausgewählt wurde, wird diese geöffnet und es wird versucht,
Daten für die anderen Felder des Bereichs auszulesen. Dabei orientiert sich OVJ am Format das im Distrikt C verwendet wird.

Normalerweise wird die Anzahl der Teilnehmer bei der Auswertung ermittelt. Sobald jedoch im Feld 'Anz. Teilnehmer manuell' eine Zahl steht, wird diese stattdessen verwendet.

Um Ergebnislisten verarbeiten zu können, muß OVJ das Format mitgeteilt werden. Das geschieht im Feld 'Muster'.
Das Muster kann entweder manuell eingegeben oder aus einer Liste ausgewählt werden. Dazu betätigt man den Button 'Muster', worauf sich ein neues Fenster öffnet.
Eine Beschreibung der Auswertungsmuster gibt es hier.
  • Über den Button 'Speichern' können die OV Wettbewerbsdaten abgespeichert werden. Dabei wird automatisch der Name des
    Wettbewerbs mit angehängten '_ovj.txt' verwendet und die Datei im projektspezifischen Unterverzeichnis unterhalb von 'config' abgelegt.
  • Über den Button 'Importieren' kann ein OV-Wettbewerb über die entsprechende _ovj.txt Datei ausgewählt werden,
    von dem die Daten des Veranstalter und des ausrichtenden OVs in die Eingabefelder übernommen werden.
    Diese Funktion ist hilfreich, falls der gleiche Veranstalter mehrere Wettbewerbe veranstaltet hat.
  • Über den Button 'Auswerten' kann der Wettbewerb separat ausgewertet werden.
  • Über den Button wird 'Abbrechen' wird das Fenster des OV-Wettbewerbs geschlossen
Manchmal muss man die OVFJ Auswertungsdatei bearbeiten, z.B. um Helfer einzutragen.
Die Bearbeitung der OVFJ Auswertungsdatei beschreibt die Möglichkeiten.
