Bearbeitung der OVFJ Auswertungsdatei


Im Kopfteil der Auswertung können Daten angegeben werden, die von OVJ automatisch in die entsprechenden Felder eingetragen werden,
siehe Format einer Ergebnisliste zur automatischen Auswertung durch OVJ

Um Helfer anzugeben gibt es zwei Möglichkeiten (jeder Helfer darf aber nur einmal auftauchen):
  1. Sie werden im Kopfteil spezifiziert, dabei müssen die anderen Schlüsselwörter gar nicht verwendet werden.
  2. Sie werden im Ergebnisteil hinzugefügt, mit Pseudoangaben zu Zeiten, Sendern etc., um das Spaltenformat einzuhalten.
    Dabei müssen sie innerhalb eines Helferabschnitts angegeben werden.
Abschnitte innerhalb der Liste werden in Anlehnung an HTML mit einer <KEYWORD>-Zeile begonnen und mit einer </KEYWORD>-Zeile beendet.
Falls die schließende </KEYWORD>-Zeile fehlt, gilt der Abschnitt bis zum Ende der Datei.

Schlüsselwort Bedeutung Kommentar
<Helfer> </Helfer> Teilnehmer in diesem Abschnitt werden als Helfer behandelt. Das Spaltenformat muss mittels Pseudowerten eingehalten werden.
<Keine Sonderpunkte> </Keine Sonderpunkte> Teilnehmer in diesem Abschnitt bekommen keine Sonderpunkte, z.B. für den ersten oder zweiten Platz als Nicht-PM Damit können Teilnehmer, welche das Laufzeitlimit überschritten haben, aber trotzdem Sonderpunkte bekommen würden, von diesen ausgeschlossen werden.
<Ignorier> </Ignorier>
<Ignoriere> </Ignoriere> 		Zeileninhalte in diesem Abschnitt werden vollständig ignoriert. Damit können z.B. OVJ Meldungen 'kein Match bei:' umgangen werden. Das gilt für alle Inhalte abgesehen von </Ignorier> bzw. </Ignoriere> Zeilen

Beispiel:
18      Irmgard Cagan           DH6CI   2.36.57 3       P31
19      Marie Cagan             SWL     2.37.01 3       P31
<Helfer>
20      Matthias Kühlewein      DL3SDO  0.00.00 0       P12
</Helfer>
<Ignorier>
vy 73 de Marc, dl5sez

Falls es in den Namensspalten Namen mit Umlauten, Leerzeichen oder anderen Zeichen gibt, welche von OVJ nicht korrekt
verarbeitet werden, so muß der jeweilige Vor- oder Nachname in Anführungszeichen gesetzt werden.
